- Unity - это Roblox для детей старшего возраста.
- Unreal Engine - AAA для "маминых бизнесменов".
- Godot - протест Unity-оппозиционеров.
- O3DE - CryEngine "на выгуле".
- Bevy - элитарный движок "у нас дома" для rust-маньяков.
- Harry's Mod - для олдов, выросших на "Полураспаде".
- id Tech 1 - конструктор странных рекордов.
- Ren'Py - magic wand "для креативных".

А Roblox... это Roblox. Песочница для самых маленьких и тех, кто так и не вырос.
